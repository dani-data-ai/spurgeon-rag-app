# CUDA Requirements:
# - faiss-gpu 1.7.4 requires CUDA 11.7 or higher
# - Check your CUDA version: nvidia-smi
# - Driver must support CUDA 11.7+ (usually driver >= 515.x on Linux, >= 516.x on Windows)
#
# Common Issues:
# 1. "ImportError: libfaiss.so: cannot open shared object file"
#    → CUDA not installed or version mismatch
#    → Solution: Install faiss-cpu instead
#
# 2. "RuntimeError: CUDA initialization: No CUDA-capable device is detected"
#    → No GPU detected or driver issue
#    → Solution: Check nvidia-smi, update drivers, or use faiss-cpu
#
# 3. "GPU available: 0" when you have a GPU
#    → CUDA version mismatch
#    → Solution: Check CUDA version and reinstall matching faiss-gpu or use faiss-cpu
#
